JavaScriptをいじってみる

(2023/03/17)

(目安 20分)

開発のスキルを身に着けるにはどうしたらよいのかは、私もわかっておりません。人によってそれぞれです。試行錯誤しながらお互いに合った方法を探しています。

JavaScriptは一般的にウェブサイトやウェブアプリの開発で使います。しかしその目的に限らずプログラミングを少し試すには便利ですので、まずはJavaScriptとしています。

解説

1

次のJavaScriptのプログラムを実行すると、単に10と表示されます。

console.log(10);

前回(lesson005)ブラウザのConsoleで実行してみましたが、忘れていたらもう一度ブラウザのConsoleから試してみてください。

2

次の2行のプログラムもやはり単に10と表示されます。

const a = 10;
console.log(a);

const a = 10 というのは10という値にaという名前の名札を貼るという意味です。

3

次の3行のプログラムもやはり単に10と表示されます。

const a = 10;
const b = a;
console.log(b);

aという名前の名札を貼られた値にbという名前の名札を貼るという意味です。aとbの2つの名札が同じ値に貼られることになります。

4

次の2行のプログラムは12と表示されます。

const a = 10;
console.log(a + 2);

5

次の4行のプログラムは、10, 12の順に2つの数字が表示されます。

let a = 10;
console.log(a);
a = 12;
console.log(a);

さきほどのconstの代わりにletになっています。

let a = 10 というのはconstのときとほとんど同じですが、aという名前の名札を後から別の値に貼り変えることができます。constはあとで貼り変えることができません。

このへんの書き方は言語によって異なります。JavaScriptを本格的に書けるようになりたいのであれば覚えておいてください。JavaScript不要であればconstやletという書き方まで覚える必要はありません。ただし、書き方が違うだけで同じ概念はどの言語にもあるので、名札の貼り方に2種類あるということは覚えておいてください。（細かいこと言うと言語によって違いはいっぱいあります。JavaScriptも本当は3つ目の書き方がありますが、ここで説明したいことではないので省略です）

問題

次の4行のプログラムを実行すると、なんと表示されますか？ブラウザで実行させずに、結果を考えてみてください。

let a = 10;
console.log(a);
a = a + 20;
console.log(a);

次のプログラムもどうなるか考えてみてください。

let a = 1;
let b = a + 2;
console.log(b);
a = 5;
console.log(b);
