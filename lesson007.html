前回の宿題の2つ目は以下のような動きになります。

let a = 1;        // 1という値にaという名前の名札を付けます
let b = a + 2;    // a + 2 を計算し、その結果である3という値にbという名前の名札を付けます
console.log(b);   // 3と表示されます
a = 5;            // 5という値にaという名前の名札を付けます。最初に名札を貼った1からは名札がなくなります。bという名札には変化ありません
console.log(b);   // bという名札は3に付いたたままですので、3と表示されます

※プログラミング言語によっては b = a + 2 の部分で3という値に名札を付けるのではなく a + 2 という式に対して名札を付ける書き方もあります。その場合は、 a = 5 のあとでは b が7になります。

参考までに、同じ内容をPythonで書くと次のようになります。

a = 1        # 1という値にaという名前の名札を付けます
b = a + 2    # a + 2 を計算し、その結果である3という値にbという名前の名札を付けます
print(b)     # 3と表示されます
a = 5        # 5という値にaという名前の名札を付けます。最初に名札を貼った1からは名札がなくなります。bという名札には変化ありません
print(b)     # bという名札は3に付いたたままですので、3と表示されます

「名札」のことをプログラミング言語によっては変数と言います。
